# Catálogo de Componentes - Ilsa Web

## Átomos (Atoms)
- **Button.astro**: Botón general del sistema.
- **NavButton.astro**: Botones específicos para la navegación.
- **SliderArrow.astro / SliderProgress.astro**: Elementos para controles de carrusel.
- **Breadcrumbs.astro**: Navegación de migas de pan.

## Moléculas (Molecules)
- **OpinionCard.astro**: Tarjeta especializada para columnas de opinión.
- **ArticleCard.astro**: Tarjeta estándar para listados de artículos.

## Organismos (Organisms)
- **Navbar.astro**: Barra de navegación principal que ensambla átomos de navegación.
- **ArticlePost.astro**: Contenedor principal para el cuerpo de artículos de WordPress.
- **RelatedSlider.astro**: Carrusel de contenido relacionado.
